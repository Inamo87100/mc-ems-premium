== Description ==

**MC-EMS Premium** is a premium add-on for [MC-EMS – Exam Session Management](https://wordpress.org/plugins/mc-ems-base/) (Base). It extends the free base plugin with unlimited exam sessions, advanced exam booking management, and enhanced certification workflows — ideal for elearning platforms, certification bodies, and universities running large-scale exam scheduling with Tutor LMS.

> **Requires MC-EMS Base** (free plugin) to be installed and active. This add-on cannot run standalone.

= What does Premium add? =

MC-EMS Premium unlocks all limits of the Base version, enabling large-scale exam management for:

* Certification bodies handling thousands of candidates
* Large elearning platforms with high exam-booking volume
* Universities with multiple exam sessions per day
* Organizations requiring advanced exam scheduling and proctor management

= Exclusive Premium Features =

**Unlimited Exam Sessions**
The Base version caps sessions at 5. With Premium, create unlimited exam sessions and manage thousands of sessions per exam — no administrative limits.

**Up to 500 Seats per Session**
The Base version limits capacity to 5 seats per session. With Premium, support up to 500 seats per session for large-group exam scheduling.

**Multiple Slots per Day**
The Base version allows 1 slot per day per exam. With Premium, create up to 999 slots per day — perfect for multi-shift certification exams.

**Advanced Booking Search**
Toggle between Basic (single-date) and Advanced (date-range) search modes. Filter by exam, candidate, and status. Optimized CSV export included.

**Enhanced Booking Management**
Superior admin interface with faster filters, bulk booking management, and extended data export options.

== Requirements ==

* WordPress 6.0 or higher
* PHP 7.0 or higher
* **MC-EMS Base** (required — must be installed and active)
* Tutor LMS (for LMS integration features)

== Installation ==

1. Install and activate **MC-EMS Base** (free, required dependency) from the WordPress Plugin Repository.
2. Download the MC-EMS Premium plugin ZIP.
3. Go to **WordPress → Plugins → Add New → Upload Plugin**.
4. Upload the ZIP and click **Install Now**.
5. Click **Activate**.
6. Premium features are immediately available — no additional configuration needed.

== Frequently Asked Questions ==

= Can I use MC-EMS Premium without MC-EMS Base? =

No. MC-EMS Base is mandatory. It is the foundation this add-on builds upon.

= Does Premium replace the Base plugin? =

No. Premium integrates seamlessly alongside the Base plugin and removes its limits. Both must be active.

= Will upgrading lose my data? =

No. MC-EMS Premium is fully backward-compatible with existing Base settings and data.
