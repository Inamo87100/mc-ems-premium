## Installation
1. Download the plugin from the official repository.
2. Extract the contents of the zip file to your `wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.

## Usage
- After activation, navigate to the plugin's settings page to configure the features.
- Follow the on-screen instructions to utilize the full potential of the mc-ems-premium plugin.
